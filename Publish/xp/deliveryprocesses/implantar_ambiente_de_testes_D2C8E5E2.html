Task: Implantar o Ambiente de Testes
Purpose
  • Criar um ambiente estável para execução dos testes de cliente.
Relationships
RolesPrimary:
  • Testador XP
Additional: Assisting:
Steps
Geral

Os testadores terão que configurar algum tipo de ambiente de hardware e/ou software para executar os testes de aceitação de cliente. O ambiente pode exigir a instalação de ferramentas de teste de software específicas ou o sistema operacional poderá necessitar de ajustes específicos de ambiente. Tente reproduzir, tanto quanto possível, um ambiente típico de usuário final, quando da execução dos testes. Os testes podem necessitar da criação de múltiplos ambientes (quando são utilizados diferentes sistemas operacionais, por exemplo).

Os ambientes de teste não são apenas para testadores, é fundamental que eles sejam colocados à disposição dos programadores. A execução dos testes de aceitação é a única forma de saber se a história está concluída e se as histórias anteriores não foram danificadas.
