Work Product Descriptor (Artifact): Código de Produção
A especificação executável do sistema que está sendo construído.
Purpose

Na XP, o Código de Produção é considerado o artefato mais importante. Ele é o único artefato de design que não pode ser substituído, pois é a única expressão completa e inequívoca das intenções de design. O código-fonte é uma especificação. Ele, junto com um compilador ou interpretador, engloba toda a semântica necessária para produzir um processo executável em computador.

Main Description

Esta definição do Código de Produção engloba o software codificado à mão, bem como modelos executáveis. O Código de Produção deve ser mantido limpo e simples, visto que é o principal veículo para comunicar a intenção de design para a equipe de programação. O código tem testes de unidade e de cliente completos. As práticas XP de design simples, programação em pares, refatoração, propriedade coletiva de código, desenvolvimento dirigido por testes e padrão de codificação suportam a criação do código.
