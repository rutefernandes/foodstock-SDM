Task: Manter o Processo no Rumo
Purpose
  • Garantir que o processo esteja sendo usado eficazmente de forma que as metas do projeto sejam alcançadas.
Relationships
RolesPrimary Performer:
  • Treinador XP
Additional Performers:
Steps
Geral

Toda a Equipe precisa ser muito focada nas metas do projeto. As práticas de desenvolvimento permitem que as metas do projeto sejam alcançadas. O foco nas metas do projeto e no uso centrando e disciplinado das práticas é um desafio constante para a equipe.

O Rastreador XP coleta os dados e os informa à equipe. Esses dados fornecem orientações para saber se a equipe está na trajetória para cumprir as metas do projeto. Os dados também monitoram o uso das práticas pela equipe.

O Treinador XP assegura que a equipe esteja recolhendo e usando os dados para garantir que as metas do processo e do projeto sejam alcançadas.
