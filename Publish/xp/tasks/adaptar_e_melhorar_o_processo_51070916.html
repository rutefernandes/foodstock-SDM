Main Description

Essa tarefa tem como o objetivo da equipe melhorar o processo que irão utilizar, adaptando-o para suas necessidades.

Steps
Geral

As equipes que usam XP são guiadas pelos Valores XP no uso das Práticas XP. As práticas XP são melhores práticas, mas elas também alavancam as vantagens das outras práticas para formar um conjunto mínimo e eficiente de práticas necessárias para entregar software de alta qualidade alinhado às necessidades dos clientes.

Em geral, as equipes serão mais eficazes no uso da XP se cada uma das práticas for utilizada o máximo possível no projeto. Na prática, isto pode ser difícil de conseguir, por várias razões, incluindo:

  • Obstáculos técnicos, tais como linguagens não-OO e código legado
  • Falta de competências, incluindo competências de programação básica, habilidades práticas, especialização no domínio e aptidões de trabalho em equipe
  • Um ambiente de cliente complexo, onde uma única fonte de histórias priorizadas seja difícil de desenvolver
  • Obstáculos Organizacionais, tais como equipes distribuídas, grandes equipes e culturas orientadas a comando e controle

À medida que a equipe usa XP, esses obstáculos afetam a sua capacidade de usar eficazmente as práticas. O Treinador XP ajuda a equipe a tratar como esses desafios irão afetar a utilização das práticas. Ele começa ajudando a equipe a manter a Coragem para enfrentar e remover esses obstáculos, limpando o caminho para as práticas serem usadas. Nos casos onde os obstáculos não puderem ser removidos, o treinador e a equipe usarão os Valores XP para guiar na adaptação das práticas.

O Treinador XP precisa participar de comunidades que compartilhem as melhores práticas no desenvolvimento de software e XP. Estas comunidades existem dentro de grandes empresas, em grupos de usuários locais e na Internet.
