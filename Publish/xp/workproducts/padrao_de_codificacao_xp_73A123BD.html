Artifact: Padrão de Codificação
Descreve as convenções a serem usadas ao se trabalhar com a linguagem de programação.
Purpose

A finalidade do Padrão de Codificação é facilitar a comunicação entre os programadores que trabalham no mesmo código base. A codificação padrão tem grande importância na XP por causa das práticas de programação em pares e propriedade coletiva de código. O objetivo é que todas as partes do código pareçam ter sido escritas pelo mesmo programador.
