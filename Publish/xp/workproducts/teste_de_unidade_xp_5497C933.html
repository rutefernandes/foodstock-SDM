Artifact: Teste de Unidade XP
Usado para garantir que uma determinada funcionalidade de um componente do sistema esteja funcionando corretamente.
Purpose

Os Testes de Unidade XP são construídos pelos desenvolvedores para garantir que todos os componentes do sistema funcionem como esperado. Os Testes de Unidade XP dão aos desenvolvedores a tranquilidade de modificar o sistema e se sentirem confiantes de que tudo ainda funciona.

Description
Main Description

Os Testes de Unidade XP são geralmente implementados no nível de classe (em linguagens OO) e testam a interface pública. Para cada classe no sistema, existe um teste de classe. Frameworks populares de teste estão disponíveis gratuitamente (tais como JUnit, CppUnit, CppUnitLite, NUnit e VBUnit). Estas são ferramentas simples que ajudam o desenvolvedor a organizar e executar testes de unidade. Estes testes direcionam o desenvolvimento da programação e informam ao programador se o código funciona como o esperado.
