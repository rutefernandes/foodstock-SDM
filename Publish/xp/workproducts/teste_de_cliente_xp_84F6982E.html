Artifact: Teste de Cliente XP
É executado no sistema para verificar se uma funcionalidade foi implementada corretamente.
Purpose

O Teste de Cliente XP demonstra a presença das funcionalidades do sistema, tal como definido pelo cliente. Eles são os requisitos inequívocos e detalhados do sistema.
Relationships
RolesResponsible:
  • Cliente XP
Modified By:
  • Testador XP
TasksInput To:
  • Executar Teste de Cliente
  • Informar Resultado do Teste de Cliente
Output From:
  • Automatizar Testes de Cliente
Key Considerations
À medida que mais e mais testes de cliente forem adicionados ao sistema, haverá uma necessidade de organizá-los de alguma forma. Normalmente, os testadores e programadores irão construir e manter algum tipo de framework de teste de cliente. Da mesma forma como acontece com o código de produção, é importante deixar o framework evoluir com as necessidades da aplicação. É muito fácil extrapolar quando da construção de um framework de teste.
